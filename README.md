🏨 Sistema de Controle de Reservas de Hotel
Sistema desenvolvido em Java para gerenciamento de reservas de hotel, aplicando conceitos de Programação Orientada a Objetos (POO).
📋 Sobre o Projeto
O sistema permite controlar reservas de quartos, gerenciando hóspedes, disponibilidade e períodos de estadia. Desenvolvido como projeto prático para consolidar fundamentos de Java e POO.
⚙️ Funcionalidades

Cadastro e gerenciamento de hóspedes
Controle de disponibilidade de quartos
Criação e cancelamento de reservas
Consulta de reservas por período

🛠️ Tecnologias

Java 11+
Programação Orientada a Objetos (Encapsulamento, Herança, Polimorfismo)
Estruturas de dados e coleções Java

🚀 Como Executar
bash# Clone o repositório
git clone https://github.com/butyrum/projeto-hotel

# Entre na pasta do projeto
cd projeto-hotel

# Compile o projeto
javac src/*.java -d out

# Execute
java -cp out Main
